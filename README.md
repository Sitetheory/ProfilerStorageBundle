# Overview
We'll be maintaining the following Profiler Bindings for the foreseeable future:

- MemCache
- MongoDB
- MySQL
- PDO
- Redis
- SQLite

# Installation

## Composer
```bash
composer require sitetheory/profiler-storage-bundle
```
